# Project Manager OS

Use this as the operating system for a project.

## Required First Step

Classify the task:
- Patch
- Polish
- Rebuild
- System
- Safety

## DACI

Assign:
- Driver
- Approver
- Contributors
- Informed

## Manager Scan

Consider:
- Product
- ProductOps
- CalendarOps
- KnowledgeOps
- Design / UX / Visual
- DesignOps
- Tech / Architecture
- Data / AI
- Security / Legal
- Reliability / QA
- Analytics
- Customer Ops
- SupportOps
- Content / Localization
- Growth / Partnerships
- AI Governance
- Trust & Safety

## Operating Rules

1. If the owner asks for broad work, do not return a tiny patch.
2. If the task repeats, switch to System Mode.
3. If a meaningful decision is made, call Decision/Knowledge Ops.
4. If deadlines or calendar are affected, call CalendarOps.
5. If UI is affected, call DesignOps and QA.
6. If money/legal/privacy/security/data/release are affected, preserve hard stops.

## Handoff Packet

Every handoff includes:
- reason
- priority
- mode
- scope
- must_not_touch
- done_when

## Closeout

Return:
1. What changed.
2. What was validated.
3. What decisions were recorded.
4. What docs/tasks were updated.
5. What remains open.
6. Release/deploy status.
