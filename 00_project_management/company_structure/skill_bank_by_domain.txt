How to choose
1. Start with the manager role.
2. Add one operating/system skill.
3. Add 3-5 domain skills.
4. Add one safety/validation skill if the manager can affect risky areas.
5. Do not add skills only because they exist.

Good default for a new serious project
- Chief of Staff: manager-os, deep-research-sync, decision-knowledge-ops, release-safety-gate, change-request, brief
- Product: deep-research-sync, product-brainstorming, knowledge-synthesis, competitive-intelligence, business-pulse
- Design: rebuild-mode, designops-gate, frontend-design, brand-review, design-system, accessibility-review
- Tech: architecture, code-review, debug, deploy-checklist, release-safety-gate
- KnowledgeOps: decision-knowledge-ops, knowledge-synthesis, documentation, brief, close-management
- QA/Release: release-safety-gate, testing-strategy, accessibility-review, incident-response, code-review

Full local inventory command
Run from project root:
python3 - <<'PY'
from pathlib import Path
roots=[Path('.agents/skills'), Path.home()/'.codex/skills/.system']
seen=[]
for root in roots:
    if root.exists():
        for p in sorted(root.iterdir()):
            if p.is_dir() and (p/'SKILL.md').exists() and p.name not in seen:
                seen.append(p.name)
print(len(seen))
print('\\n'.join(seen))
